## ERC 환경구성

#### 오픈제플린 설치
```
$ yarn add @openzeppelin/contracts
```

#### 트러플 설치
```
$ yarn add truffle
```

#### 가니쉬 설치
https://github.com/trufflesuite/ganache-cli
```
$ yarn add ganache-cli
```

#### 이더월렛 프로바이더 설치
```
$ yarn add @truffle/hdwallet-provider
```
